DNS メモ

インストール

Ubuntu

ldnsutilsパッケージをインストールする

apt-get install ldnsutils

問い合わせ

drill example.com

dig example.com

DNSサーバーを指定する

drill example.com @8.8.8.8

dig example.com @8.8.8.8

ルートサーバーから順に問い合わせる

drill -T example.com

dig +trace example.com

MXレコード問い合わせ

drill mx example.com

dig example.com mx
