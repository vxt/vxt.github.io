環境変数を確認する方法

Linux

envコマンド

echo $VAR1 #VAR1を表示

特定のプロセスの環境変数を確認する

cat /proc/$PID/environ

環境変数を変更してプログラムを起動する

env -i COMMAND #環境変数をクリアして起動

env VAR1=100 VAR2=200 COMMAND #環境変数をセットして起動

env -u VAR1 -u VAR2 COMMAND #環境変数を消して起動

Windows

setコマンド

引数無しで全ての環境変数を出力する

set V #Vから始まる環境変数を出力する

echo %VAR1% #VAR1を出力する

set VAR1=100 #環境変数をセット
