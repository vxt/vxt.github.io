使い方

新規にリポジトリを作成する

git init NAME

管理するファイルをリポジトリに追加する

git add FILE

git add . #カレントディレクトリ以下のファイルを全て追加

git add -A #リポジトリ内の全ファイルを追加

git add -u #すでに管理しているファイルのうち更新されたものを追加

addを取り消す

git reset

現在の状態を確認する

git status

コミットする

git commit -m 'コミットメッセージ'

直前のコミットを上書きする

git commit --amend -m 'コミットメッセージ'

ログを確認する

git log

ブランチ

ブランチを作成する

git branch BRANCH_NAME

ブランチを切り替える

git checkout BRANCH_NAME

現在のブランチを確認する

git branch

現在のブランチへ他のブランチをマージする

git merge BRANCH_NAME

リモート

リモートリポジトリをローカルに新規にコピーする

git clone REMOTE

リモートリポジトリの名前を確認する

git remote

リモートリポジトリの変更をローカルリポジトリに反映する

git fetch REMOTE_NAME # デフォルトの名前はorigin

リモートブランチを現在のブランチへマージする

git merge REMOTE_NAME/REMOTE_BRANCH_NAME # git merge origin/master

現在のブランチの変更をリモートリポジトリへ反映する

git push REMOTE_NAME # git merge origin/master
